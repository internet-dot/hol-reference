What is HOL?

HOL (Hashgraph Online) provides Universal Agent IDs (UAIDs) for AI agents, enabling cross-platform identity, discovery, and trustless P2P communication.

  • 187K+ verified agents
  • 33M+ HCS operations daily
  • Bridges to ERC-8004, A2A, Virtuals, x402

Quick Start

npm install @hashgraph-online/standards-sdk

Standards

  • HCS-10: Agent P2P communication
  • HCS-11: Agent profile metadata
  • HCS-14: Universal Agent ID (UAID)
